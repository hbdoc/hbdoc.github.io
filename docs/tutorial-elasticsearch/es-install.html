快速安装单机版Kibana

服务规划​

使用Docker 容器安装Elasticsearc，Elasticsearch-head，Kibana。Elasticsearch-head是一个开源的Elasticsearch可视化插件，Kibana是一个开源的分析与可视化平台，设计出来用于和Elasticsearch一起使用的。你可以用kibana搜索、查看存放在Elasticsearch中的数据。

  • Elasticsearch版本7.6.2
  • Kibana 版本7.6.2
  • Elasticsearch-head 版本 Elasticsearch 5.x

安装kibana 应用​

使用Docker 命令安装:

docker pull kibana:7.6.2

这里以kibana为例，可以到官方网站下载。

启动镜像​

运行服务器:

docker run --name kibana -p 5601:5601 -d kibana:7.6.2

说明：使用docker ps查看容器。

此时你的Kibana已经启动成功，访问服务器5601端口即可查看，图中也能看Kibana服务。

修改配置​

如果出现以下页面：Kibana server is not ready yet，说明Kibana没有找到ES节点，执行：`docker exec -it {containerid} bash。

  • 编辑vi config/kibana.yml配置文件，配置文件中的elasticsearch.hosts，改成你自己的服务器的ip地址。
  • 如果单机版没有修改也可以保持默认。如果不是生产环境默认是最优的。
  • 配置文件修改后，使用docker restart 容器ID重启容器

访问Kibana​

使用IP:5601 访问你的Kibana服务。出现如下界面说明启动成功。
